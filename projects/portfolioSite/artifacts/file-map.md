# file-map.md

除外: `.git/`（バージョン管理メタデータ）、`node_modules/`（未生成だが対象外）。

- AGENTS.md — LLM エージェント運用契約（Plan 承認必須、ハードコード禁止など）。
- .gitignore — 生成物・キャッシュ（.next/.turbo/.vercel/out/coverage 等）、ログ、env、OS/エディタ一時ファイルを除外する設定。.next と site/.next を明示し追跡除外。
- scripts/bootstrap.mjs — /artifacts 配下の必須ファイル・テンプレートを生成するブートストラップスクリプト（git 操作なし）。
- scripts/update-gitignore.mjs — 現在の Workset GID を .gitignore に unignore 追加する補助スクリプト。
- scripts/migrate-append-only-history.mjs — 旧 Workset/成功/失敗ディレクトリを append-only JSONL（三種）と spec.md に統合し、_archive へ退避する移行スクリプト。
- scripts/append-history.mjs — JSON 文字列を指定ファイルに1行追記するための簡易アペンダ。
- scripts/check-history-append-only.mjs — history.*.jsonl に削除/書換が無いか git diff で検査するスクリプト。
- scripts/derive-index.mjs — history.edits.jsonl から _timeline.md / _index.json を再生成するスクリプト。
- docs/Portfolio_Requirements_v1.1.md — ポートフォリオ要件定義書。
- mockup/mockup.tsx — ポートフォリオ UI モック（デザイン参照用）。
- mockup/FolderInteraction.tsx — モックのフォルダインタラクション部品。
