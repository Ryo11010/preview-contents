社内プレビューの入口

Xserver 上で `/projects/{project}/{env}/` に配置したビルド成果物をそのままプレビューできます。Basic認証とHTTPSを前提にシンプルな一覧を表示します。

一覧に出したいプロジェクトだけを `previews.json` に記載してください。存在しない URL は 404 になります。

アップロード手順（最短）

  • `public_html/` に: index.html / .htaccess / previews.json / maintenance.html / errors/404.html / errors/500.html
  • `public_html/projects/` にローカルの `/projects` を丸ごとアップロード（含: projects/.htaccess）。
  • URL: `https://preview-contents.site/projects/{project-key}/{environment}/` または `https://preview-contents.site/projects/{project-key}/`
