- docs/PreviewContentsSite_Requirements_v1.0.md — 要件定義書 v1.0。
- docs/PreviewContentsSite_Usage.md — 利用ガイド（/projects ドラッグ運用の手順）。
- errors/404.html — 404 カスタムページ。

- index.html — プレビュー一覧（previews.json を読み込む）。
